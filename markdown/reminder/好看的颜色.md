# 颜色收藏

1. ##红

   * 淡粉红色: 0x d8c6b8
   * 淡红: 0x ea586d

2. ##黄(橙)

   * 淡橙红色: 0x ffcc99
   * 亮橙色: 0x FF6600
   * 橙色: 0x FFA500
   * 暗橙色(适合做符号的颜色): 0x cc7832

3. ##绿

   * 墨蓝绿色: 0x 9eb9bc
   * 抹茶绿: 0x 94ae9a
   * iterm2终端浓墨绿色: 0x 083541

4. ##蓝(青)

   * 天青色(光标色): 0x 3eccdc
   * 淡蓝色: 0x c0ddf8
   
5. ##紫

   * 墨紫色: 0x 663366
   * 淡棕色: 0x 993333
   * 暗紫色(好看的终端的背景色): 0x 2c0d23
   * 暗紫红色(ubuntu终端默认色): 0x 35182b

6. ##白

   * 白灰色，适合做字体的前景色: 0x d6d7d7
   * 一元、二元符号色: 0x D3D3D3

7. ##灰(白)

   * 淡灰色，适合与当背景: 0x 4b4b4b 
   * 银灰色: 0x CCCCCC
   * clion边框灰: 0x 3c3f41
   * clion行号大纲框灰: 0x 313335
   * clion编辑页面灰灰: 0x 2b2b2b
